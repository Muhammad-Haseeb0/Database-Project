# NAMS Database Management System

## Overview
NAMS (National Agriculture Management System) is a relational database project designed to manage agriculture-related operations including farms, crops, resources, water supply, buyers, purchases, and sales.

This project focuses on database design, relationship modeling, SQL implementation, and efficient data management for an agriculture management system.
## Features
- Database schema design
- Entity Relationship Modeling (ERD)
- SQL table creation
- Complex SQL queries
- Data relationships using primary and foreign keys
- Data management for agriculture operations
