# NAMS Database Management System

## Overview
NAMS (National Agriculture Management System) is a relational database project designed to manage agriculture-related operations including farms, crops, resources, water supply, buyers, and sales.

## Features
- Database schema design
- Entity Relationship Modeling
- SQL table creation
- Complex queries
- Data relationships using primary and foreign keys

## Technologies Used
- MySQL
- SQL

## Database Entities
- FARM
- OWNER
- LAND
- CROPS
- RESOURCES
- SALES
- PURCHASES
- BUYER

## Concepts Implemented
- Primary Keys
- Foreign Keys
- Joins
- Constraints
- Normalization
